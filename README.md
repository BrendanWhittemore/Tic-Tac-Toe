# Tic-Tac-Toe

## Project Description
A simple command-line game of tic-tac-toe.

## Getting Started
```
git clone https://github.com/BrendanWhittemore/Tic-Tac-Toe.git
cd Tic-Tac-Toe
make
tictactoe
```

## Objectives
This project was completed as a C language refresher.
